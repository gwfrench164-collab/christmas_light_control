# ESP32 Pinout Reference

This document lists all pin assignments for the Christmas Light Controller project.

## DHT22 Sensor
| Signal | GPIO |
|--------|------|
| Data   | **14** |

## Relays
| Relay | GPIO |
|-------|------|
| 1     | 13   |
| 2     | 25   |
| 3     | 26   |
| 4     | 27   |
| 5     | 32   |
| 6     | 33   |
| 7     | 15   |
| 8     | 12   |

## VS1053 Audio Decoder
| Function | GPIO |
|----------|------|
| MISO     | 19   |
| MOSI     | 23   |
| SCK      | 18   |
| DREQ     | 4    |
| XRST     | 21   |
| XCS      | 5    |
| XDCS     | 22   |

---

### Summary by Function
- **Relays:** 12, 13, 15, 25, 26, 27, 32, 33  
- **Audio (VS1053):** 4, 5, 18, 19, 21, 22, 23  
- **Sensor (DHT22):** 14
